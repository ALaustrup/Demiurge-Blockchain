## âš ï¸ Common Issues

### Issue: "Missing Modules" Warning

**Solution:**
1. Close Editor
2. Delete `Binaries/` and `Intermediate/` folders
3. Regenerate Visual Studio files
4. Rebuild from Visual Studio
5. Reopen Editor

### Issue: C++ Compilation Errors

**Check:**
1. Visual Studio 2026 C++ tools installed
2. Windows SDK installed
3. All `#include` paths correct
4. Module dependencies in `*.Build.cs` files

### Issue: Blueprint Classes Not Found

**Solution:**
1. Ensure C++ code compiled successfully
2. Restart Editor
3. Check `*.generated.h` files exist
4. Verify `UCLASS(Blueprintable)` macro present
