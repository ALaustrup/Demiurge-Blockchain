# Qor Installer

The gateway to the Demiurge-Blockchain. Setup and configuration tool for new users.

## Purpose

Qor Installer guides users through:
1. Creating their Qor ID
2. Setting up their wallet
3. Connecting to the Pleroma network
4. Initial CGT acquisition
